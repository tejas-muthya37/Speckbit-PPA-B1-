










easyBuy is a GRAPHICAL USER INTERFACE (GUI) E-Commerce Application that lets the program communicate with the user in the form of single mouse click / double mouse click.

easyBuy is an E-Commerce Application that has a vast inventory with selections ranging from Apparels, Accessories to Books and Video Games. The user can add the desired products
onto his/her cart by clicking once on the name of the product.

The user can checkout and place his/her order after the adding of items onto the cart and entering his/her details. 

The application generates an invoice bill with the date, customer's name, products chosen and the GRAND TOTAL.
